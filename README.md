# Jani Bunka — Personal Website

This is my personal portfolio website hosted using GitHub Pages.

## Live Site
https://janibun.site

---

## About the Project

This website is a custom-built personal space showcasing my work, design style, and a bit about me.  
It is designed and developed from scratch to reflect my creative identity and experimentation with interactive web animation and modern front-end techniques.

The site focuses on:
- Clean, minimal visual design
- Smooth scroll-based interactions
- Custom animations and SVG path drawing effects
- A responsive layout optimized for both desktop and mobile

---


## Built With

- HTML5
- CSS3
- JavaScript (Vanilla)
- SVG animations
- GitHub Pages (hosting)
- Claude (debugging)

---

## Project Structure

Simple static site structure:
- `/index.html` — main page
- `/styles.css` — styling
- `/script.js` — interactions & animations
- `/assets` — images and media
